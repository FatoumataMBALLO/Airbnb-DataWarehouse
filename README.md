# 🏡 Airbnb Data Warehouse

## 📖 Présentation

Ce projet consiste à construire un Data Warehouse à partir des données Airbnb.

L'objectif est de mettre en œuvre un pipeline ETL complet en Python, de modéliser les données selon un schéma en étoile et de créer un tableau de bord Power BI.

---

## 🚀 Technologies utilisées

- Python
- Pandas
- SQLite
- SQL
- Power BI
- Git
- GitHub
